创建活动  ¶

创建活动
POST/event/create

新建一个活动

Example URI

POST http://qa.www.eventdove.com/api/event/create
URI Parameters
HideShow
title
string (required) Example: 世界互联网大会

活动名称

content
array (required) 

活动内容是一个数组

logoUrl
string (required) Example: https://ss0.bdstatic.com/5aV1bjqh_Q23odCf/static/superman/img/logo/logo_white_fe6da1ec.png

logo地址

bannerUrl
string (required) 

banner的url地址

startTime
string (required) Example: 2016-12-27T17:31:04.000Z

活动开始时间

endTime
string (required) Example: 2016-12-28T17:31:12.000Z

活动结束时间

isPublic
boolean (required) Example: true

是否公开活动

askPromotion
boolean (required) Example: true

是否需要宣传

smsNotice
boolean (required) Example: true

是否发送短信通知 + Default: false

userId
string (required) Example: 123456789

创建者用户id

organizers
array (required) 

主办方数组

categories
array (required) Example: [1,2,3]

活动类别 是一个数组,可以传入多个类别

Choices: 1 2 3 4 5 6 7 8

keyWords
array (required) Example: ["北京会唐网","会鸽网"]

活动关键词

country
string (required) Example: 中国

国家

province
string (required) Example: 北京

省

city
string (required) Example: 北京

城市

zipCode
string (required) Example: 121000

邮编

detailedAddress
string (required) Example: 北京市朝阳区朝阳路泰禾文化大厦A座4层会唐网

详细地址

lng
number (required) Example: 123.1

经度

lat
number (required) Example: 231.3

纬度

onlineAddress
boolean (required) Example: false

是否是线上活动

geohash
string (required) Example: wxaddjd

geohash

paymentAccountIds
string (required) Example: ['12344441','123444222']

支付帐号数组

paymentPriceUnit
string (required) Default: yuan Example: yuan

支付单位yuan/dollar

Choices: yuan dollar

onsite
boolean (required) Example: false

是否支持现场缴费

transfer
boolean (required) Example: false

是否支持银行转账,即线下收款

basePrice
number (required) Example: 100

基础服务费(服务费低于该额度以该额度计) 默认 0: 无

percent
number (required) Example: 5

票服务费比例(占票价百分比) 默认为 0%

maxFee
number (required) Example: 200

最高服务费(服务费超过该额度以该额度收取) 0: 无

isCollectAttendees
boolean (required) Default: false Example: true

是否收集参会者信息,默认true收集,false为只收集购票者信息

Choices: true false

tickets
array (required) 

门票

collectItems
array (required) 

表单收集项
